
# ScriptUtils

## About

A multi-purpose discord bot written in python

While made to work out-of-the-box, there will be an attempt to let users modify the bot for their own personalized use - This idea might be dropped in the future due to the estimated scale of the project, but it is currently considered.

## Modules

- Moderation - Moderation Commands and Utilities
- AutoMod - Custom Auto Moderation
- AntiRaid - ScriptUtils Anti Raid
- Developer - Developer Commands + Scripts
- Change Log - Version Control
- Console - PM/DM CLI
- Logs - Highly customisable logging for almost everything
- Utils - Tools and Utilities to help with certain tasks
- Tickets - Customisable ticketing and form system
- Economy - Simulated virtual server economy
- Levels - Advanced activity-based leveling system
- suTaskPoints - A point / level reward system for completing challenges or tasks
- suBackup - Server backups
- suDatabase - Server database / DB Admin
- suPerms - ScriptUtils Permission System
